# Overview
- This lab documents the configuration of a Domain Controller in a virtualized setup. The goal is to prepare a fully functional DC with all of the necessary services and features for client machines to join the domain. 

# Environment
- Windows Server 2025 (Evaluation)
- Desktop Experience
  
# Virtualization
- Hypervisor: VMware Workstation
- vCPUs: 2
- Memory: 4 GB
- Storage: 80 GB (virtual disk)

# Network
- Adapter type: NAT
- IP Configuration: Static
